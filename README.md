## 開発環境

 - [Node.js](https://nodejs.org/) v0.12.x


## インストール

```sh
$ git clone git@github.com:mfnewwind/newwind.git
```


## 起動
以下は、プロジェクトのディレクトリに移動して作業してください。

```sh
$ npm install
$ npm start
```
